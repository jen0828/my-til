
# my-til

Personal [TIL](https://github.com/jen0828/til)/ tech-blog built with Next.js 13

## Technologies used
* TypeScript
* Next : v13.2.4
* React : v18.2.0
* Next 13 App Router
* React Server Components
* Server-side rendering
* SSG
* Tailwind CSS

## To run next app

```bash
$ git clone https://github.com/jen0828/my-til.git
$ cd my-til
$ npm install
$ npm run dev
```
